## 🔧 技术细节

### CSS 架构

```
custom_css
├── 全局样式
│   ├── 容器配置
│   ├── 字体系统
│   └── 最大宽度
├── 组件样式
│   ├── Logo 容器
│   ├── 标签页导航
│   ├── 统计卡片
│   ├── 按钮
│   └── 输入框
├── 交互效果
│   ├── 悬停动画
│   ├── 焦点状态
│   └── 过渡效果
└── 响应式
    ├── 移动端适配
    └── 媒体查询
```

### 设计模式

1. **渐变背景模式**
   ```css
   background: linear-gradient(135deg, color1 0%, color2 100%);
   ```

2. **卡片悬停模式**
   ```css
   transition: all 0.3s ease;
   &:hover {
       transform: translateY(-Xpx);
       box-shadow: enhanced;
   }
   ```

3. **圆角系统**
   ```
   小: 10px  (按钮、输入框)
   中: 15px  (卡片)
   大: 20px  (横幅、Logo区)
   ```

4. **阴影系统**
   ```
   轻: 0 2px 10px rgba(0,0,0,0.05)
   中: 0 4px 15px rgba(0,0,0,0.1)
   重: 0 8px 20px rgba(102,126,234,0.3)
   ```
