# 🎨 Another Me - 界面视觉效果预览

## 配色方案

### 主色调
```
主品牌色 (Primary):   #667eea (靛蓝)
辅助色 (Secondary):   #764ba2 (紫色)
强调色 (Accent):      #f5576c (粉红)
```

### 功能色
```
成功 (Success):       #27ae60 (绿色)
警告 (Warning):       #e74c3c (红色)  
信息 (Info):          #3498db (蓝色)
中性 (Neutral):       #f8f9fa (浅灰)
```

### 渐变组合

#### 主渐变
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
/* 用于: Logo区域、主标题横幅、选中标签页 */
```

#### 统计卡片渐变
```css
/* RAG 文档 */
background: linear-gradient(135deg, #ffecd2 0%, #fcb69f 100%);

/* 知识来源 */
background: linear-gradient(135deg, #a1c4fd 0%, #c2e9fb 100%);

/* 对话记忆 */
background: linear-gradient(135deg, #fbc2eb 0%, #a6c1ee 100%);

/* 记忆来源 */
background: linear-gradient(135deg, #f093fb 0%, #f5576c 100%);
```

## 响应式适配

### 桌面端 (> 768px)
```
• 最大宽度: 1400px
• Logo: 180px
• 卡片: 并排显示
• 间距: 标准
```

### 移动端 (≤ 768px)
```
• 容器内边距: 10px
• Logo: 120px
• 卡片: 垂直堆叠
• 字体: 自动缩放
```

## 设计细节

### 圆角系统
```
小圆角:  10px  (输入框、小按钮)
中圆角:  15px  (卡片、容器)
大圆角:  20px  (Logo区域、横幅)
```

### 阴影系统
```
轻阴影:  0 2px 10px rgba(0,0,0,0.05)   (信息卡片)
中阴影:  0 4px 15px rgba(0,0,0,0.1)    (统计卡片)
重阴影:  0 8px 20px rgba(102,126,234,0.3) (主要元素)
```

### 字体大小
```
超大标题: 2.5em  (主标题)
大标题:   1.8em  (页面标题)
中标题:   1.2em  (副标题)
正文:     1em    (普通文字)
小字:     0.9em  (辅助信息)
```
